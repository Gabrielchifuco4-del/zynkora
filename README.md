# Zynkora - Plataforma de Tarefas

Projeto de plataforma Zynkora para simular tarefas, pacotes VIP, convites e bônus de 20%.

## Estrutura

- index.html : layout da plataforma
- style.css : estilos principais
- script.js : lógica de navegação, pacotes, tarefas, convites
- assets/ : imagens, logo, ícones

## Como testar localmente

1. Clone este repositório
2. Abra `index.html` no navegador
3. Teste navegação, pacotes, tarefas, depósitos e convites
